# moblox commands

- Change Rank

Usage:

```

!changerank [username] [rank name] (NOT ID BUT RANK NAME)

```

- Demote

Usage:

```
!demote[username];
```

- Promote

Usage:

```
!promote[username];
```

- Exile

Usage:

```
!exile[username];
```

- Get Info

Usage:

```
!info[username];
```

- Join Request

Usage:

```
!joinrequest [username] [true or false]
```

- Shout

```
!shout [message or none to clear]
```

- Help

```
!help
```
